# 📈 Stock Price Analyzer

A machine learning web app to analyze stock data and predict next-day prices.

## Features
- Technical Indicators (RSI, MACD, MA)
- Next Day Price Prediction
- Buy/Sell Signal
- Candlestick Chart
- CSV Upload
- Interactive Dashboard

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
